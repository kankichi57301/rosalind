ros_[問題の略号2\~4文字].ss (racket source)

使い方  
\>racket  
\>\>(enter! "ros_[問題の略号2~4文字].ss")  
\>\>\>(ros_[問題の略号2\~4文字] [引数])  
引数は省略可　そのときはdata/rosalind_[問題の略号2\~4文字].txtが対象  
指定したときはdata/rs_[問題の略号2\~4文字][連番].txt　を処理する  
#lang racketで始まっていない一部のソースはenter!じゃなくてloadすること
